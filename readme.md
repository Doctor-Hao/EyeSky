# Проект Дроны

## Требования
- Docker версии выше 25.0

## Компоненты
- Python
- Vue3
- Minio
- Kafka
- Nginx

## Запуск проекта
1. Для запуска проекта используйте следующую команду:

```
docker compose up -d --build
# ИЛИ
docker-compose up -d --build
```
2. Откройте в браузере http://localhost
